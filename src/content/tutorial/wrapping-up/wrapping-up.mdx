### Recap
In this tutorial, you created a React app with the Carbon Design System. You learned the ins and outs of using Carbon components, while becoming familiar with web development best practices along the way. Specifically, you learned:

1. [**Installing Carbon**](/tutorial/react-step-1)
   - Created a web app with the UI shell component.
1. [**Building pages**](/tutorial/react-step-2)
   - Built out pages with the grid and various components.
1. [**Using APIs**](/tutorial/react-step-3)
   - Populated the data table with an external data source.
1. [**Creating components**](/tutorial/react-step-4)
   - Extended Carbon by creating your own components.
1. [**Deploying to IBM Cloud**](/tutorial/react-step-5)
   - Hosted your app in a production environment.
